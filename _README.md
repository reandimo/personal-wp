# personal-wp

A personal WordPress project featuring a basic block-oriented theme.

## Contents

### Personal Block Theme

A minimal starter theme built with WordPress Full Site Editing (FSE) capabilities. Located in `themes/personal-block-theme/`.

**Features:**
- Full Site Editing support
- Block-based templates and template parts
- Global styles configured via theme.json
- Responsive layout with constrained and wide widths
- Custom color palette and typography
- Basic templates for common page types

**Getting Started:**
1. Copy the theme folder to your WordPress installation's `wp-content/themes/` directory
2. Activate the theme in WordPress Admin (Appearance > Themes)
3. Customize using the Site Editor (Appearance > Editor)

For detailed information, see the [theme's README](themes/personal-block-theme/README.md).

## Requirements

- WordPress 6.0 or higher
- PHP 7.4 or higher
